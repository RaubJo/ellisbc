# Ellis Baptist Church Website

This is the codebase for Ellis Baptist Church's website.

## Links
- Production: https://www.ellisbc.org
- Staging: https://dev.ellisbc.org
- Development: localhost:3000 or localhost:8788

## Stack
- Astro
- React
- Cloudflare Wrangler
- EmailJS
- Framer Motion
